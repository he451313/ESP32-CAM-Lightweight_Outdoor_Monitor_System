#  ESP32-CAM 低功耗大範圍監控系統

這是一個針對戶外環境設計的監控專案。利用 ESP32-CAM 的間隔採樣，實現在無網路環境下的長效監測。

## 核心技術亮點
- **間隔採樣**：系統每 2 秒喚醒一次進行偵測，其餘時間進入輕度睡眠以節省電力。
- **偵測策略**：
  - **偵測模式**：使用 QQVGA (160x120) 灰階影像進行極速比對。
  - **儲存模式**：觸發後切換至 VGA (640x480) 彩色 JPEG 進行高畫質存檔。
- **暖機補償機制**：透過廢圖丟棄，解決了 OV2640 頻繁喚醒導致的色偏問題。
- **開發/部署模式**：
  - 開發模式：透過 UART (921600 baud) 傳送至 PC 虛擬 SD 卡。
  - 部署模式：切換改為寫入實體 MicroSD 卡。

## 🛠️ 使用說明
1. 修改 `main.cpp` 中的 `USE_REAL_SD_CARD` 切換儲存模式。
2. 若使用虛擬 SD 卡，請執行電腦端的 `mock_sd_card.py` 接收照片。
3. 調整 `diff_threshold` 與 `motion_limit` 以適應不同的戶外光影環境。
